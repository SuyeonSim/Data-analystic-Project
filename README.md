# 📊 이커머스 객단가 최적화 및 전략 분석 프로젝트

## 📌 Project Overview
본 프로젝트는 프로모션 종료 후 발생한 전월 대비 객단가(AOV) 11,000원 하락 문제를 해결하기 위해 진행되었습니다. 신규 고객 유입 패턴과 결제 구간별 상품 조합을 분석하여, 브랜드의 수익성을 개선하고 차기 프로모션을 위한 전략적 가이드라인(쿠폰/사은품 허들)을 도출하는 것을 목표로 합니다.

## 🏃 분석 프로세스 (Pipeline)
01. Data Preprocessing
24,000여 건의 Raw Data 정제

주문번호 기반 주문일자 파싱 및 최근 1년 시계열 데이터 복원

실결제 금액 기준 유효 데이터 선별 및 고객 유형(회원/비회원) 분류

02. New Member Analysis
1월 상시 기간 신규 가입 고객의 구매 성과 측정

저단가 유입 상품(100원딜 등)이 신규 고객의 객단가 하락에 미치는 영향 분석

03. AOV (Average Order Value) Analysis
최근 1년 결제 금액 구간별 주문 분포 시각화

누적 비중 80% 지점 파악을 통한 전략적 사은품/무료배송 허들 산출

04. Bundle Purchase Pattern Analysis
금액 구간별 빈번한 상품 조합(Pairs) 추출

합구매(Bundle) 패턴 분석을 통한 교차 판매(Cross-selling) 및 세트 상품 기획 제언

## 💡 Key Insights & Strategy

문제 진단: 100원딜 등 저단가 유입 비중(약 36%)이 높아지며 전체 객단가 하락을 견인함.

해결 방안: 3~7만원 주력 구간 고객을 7~10만원 구간으로 유도하기 위한 'N만원 이상 구매 시 사은품' 전략 및 합구매 유도 상품 배치 제언.

전략 수립: 분석 기반 쿠폰 허들(5/7만원) 및 사은품 증정 기준(10/13만원) 도출.
